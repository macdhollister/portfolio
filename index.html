Projects

Seattle Coffee Search

GitHub Repo

Seattle Coffee Search was my first team-built web application. The idea of SCS is to provide an easy way to find coffee in a certain neighborhood of Seattle. Once a user selects a neighborhood, the nearest 50 coffee shops appear on the map. If the user wants to see more, the map allows for clicking in any location and loading the nearest 50 coffee shops to wherever they clicked.
My responsibilities in this project included creating the Google Maps API call, researching data for neighborhood boundaries, writing and implementing functions to place map markers, and generally overseeing the clean merging of parts written by each group member.
